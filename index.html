1.1 第一章定位

第一章「入口水巷」是《灰燈寨》的完整可玩 DEMO。目標不是展示所有遊戲機制，而是讓玩家在 55-70 分鐘裡經歷一次完整的情感弧線——從震撼到探索到專注到釋放到驚悚——並在最後一秒種下「我想繼續往下玩」的鉤子。

DEMO 結束時玩家應該帶走三個印象：灰燈寨是一個有人住的真實地方、阿徹是一個我想跟著走下去的角色、那個「不存在的樓層」是我必須知道的秘密。

1.4 語言設計原則

阿徹是中英混血兒。父親為英裔技術工人，母親為城寨本地居民。主要語言為英文，在以下情境切換為中文（粵語）：

  • 本地事物、食物名稱：「煲仔飯」「阿灰」「老莫」「士巴拿（扳手）」
  • 情緒觸發、城寨記憶：結尾切換「給個準話」（粵語）
  • 與老祁相關的一切：老祁是用粵語帶他入行的
  • 憤怒或粗口：粵語更有力量

台詞調性為黑色幽默——用嘲諷保護自己，用玩笑消化恐懼。但在真正重要的時刻嘴會閉上。

1.5 美術風格總則

每個場景的設備都是兩個時代的混合體——殖民時期安裝的原始設備帶有蒸氣龐克的維多利亞式精緻美學，被城寨居民用柴油龐克的方式改裝、擴建。兩者在同一台設備上共存。

蒸氣龐克元素（殖民層）

黃銅管線 + 花紋法蘭 + 壓力錶 + 蝶形閥 + 花體英文 + 維多利亞裝飾鑄鐵 + 精密齒輪

柴油龐克元素（城寨層）

粗暴鑄鐵外殼 + 大型鉚釘 + 裸露齒輪鏈 + 無裝飾功能至上 + 手寫膠帶標籤

賽博龐克元素（資訊層）

CRT 螢幕雪花屏 + 系統終端 + 報修工單數位格式 + 偶爾閃現的數據殘影

環境視覺（美術）

全黑 90 秒。不是選單的黑——是有空間感的黑暗。VR 空間追蹤正常運作（轉頭有聲音方位變化）。左下角載入指示：黃銅錶殼圓形壓力錶，直徑約 8cm，花體英文刻度「0-100 PSI」，纖細銅指針末端有裝飾性箭頭。錶面玻璃有一道細裂紋從2點鐘方向延伸到中心。指針從0緩慢攀升到80-90（不到100——系統從未完美）。

聲音設計（程式）
  • 0-5秒｜水滴層：左前方上方45度約10-15m。不規則間隔（2s/3.5s/1.8s/4s隨機）。每滴結構：清脆嗒（50ms）→金屬共振尾音（800ms）→迴音沿垂直結構上傳2-3秒消失
  • 5-15秒｜風層：右後方4點鐘→穿過頭頂→左前方10點鐘。被金屬管道切割的口哨質感（800-1200Hz泛音）
  • 15-30秒｜機械層：30-50Hz極低頻從腳下。手柄同步脈搏開始（1.2Hz，振幅從0漸增到20%）
  • 30-50秒｜結構層：鋼纜嗡聲（左上20m）＋鐵皮吱嘎（右5m）＋遠處搖晃嗒嗒嗒（正上30m+）＋蒸氣排放嘶嘶（右前15m，每20秒一次持續1.5秒）
  • 50-70秒｜人的痕跡：極遠處（50m+感覺）粵語收音機雜訊約5秒後消失
  • 70-90秒｜五層同時存在。壓力錶到頂。直接進入遊戲不跳選單
觸覺規格（程式）

第30秒手柄開始低頻脈搏。頻率1.2Hz（比人類平靜心率略慢）。振幅5秒內從0%漸增到20%。這是灰燈寨的「心跳」——中央升降梯主引擎殘餘振動穿過金屬骨架傳導。此脈搏將貫穿全遊戲。

第二層 15~50m 中景區：吊樓窄橋曬台。出前一丁空包裝當晾夾。藍色人字拖掛電線上。窗口光混合（鎢絲暖黃/燭火搖曳/LED冷白）。霓虹殘骸「廿四小時維修 24HR REPAIR」。某層外牆直徑超半公尺蒸氣主管——銅綠斑駁帶裝飾性渦卷鑄鐵支架。管線上被居民直接蓋棚屋（利用餘溫）。某層白色被單在晾——乾淨的今天洗的。

第三層 50m+ 霧區：巨大黑色剪影某些傾斜。蒸氣排放口噴白色蒸氣柱——像城寨從毛孔冒汗。結構微微搖晃應力聲。

封鎖線柵欄上：中英雙語公告「灰燈寨封鎖區域 第3號通告」。馬克筆「裡面還有人住啊幹」。褪色尋人啟事。柵欄底部：新鮮腳印 + 剪斷的鐵絲網（剪口閃亮沒鏽蝕——有人比阿徹先進去了）。

龐克元素（企劃）

蒸氣龐克（殖民層）：黃銅管線＋花紋法蘭＋壓力錶＋蝶形閥＋花體英文＋維多利亞鑄鐵裝飾。柴油龐克（城寨層）：粗暴鑄鐵＋大型鉚釘＋裸露齒輪＋無裝飾＋手寫膠帶標籤。兩套管線纏繞在一起——精緻黃銅被粗暴鑄鐵包裹穿越寄生。殖民優雅被城寨野性吞噬但沒消化。

程式備註

30秒自由凝視——不催促不彈UI。玩家可以自由轉頭觀看。此時無傳送移動功能。

玩法操作（程式）

互動A 撕塑膠帶 ×2：手進入15cm互動範圍→微弱振動脈衝暗示可互動→握拳抓取→向旁拉（帶子繃緊視覺變形＋吱吱聲＋手柄阻力漸增）→啪斷裂（阻力突然消失）。第一條較新有彈性。第二條較舊一拉就斷。兩條阻力不同——教學「材質影響觸感」。

互動B 推柵門：雙手各抓橫桿（冰冷金屬管觸感直徑約3cm表面鏽蝕粗糙）→前半段巨大阻力（門底鏽蝕摩擦地面——尖銳金屬刮水泥聲＋強持續震動＋碎鏽片散落積水棕色渾濁）→臨界點鏽蝕壓碎（阻力驟減60%）→後半段順暢→撞門檔（沉悶碰撞＋門框微振）。

連帶反應 罐頭掉落：柵門震動→鐵絲鬆→維他奶鋁罐（紅白經典包裝壓扁但商標能認）脫離。音效序列：0.0s叮（高頻碰撞）→0.3s噹（中頻空心共振）→0.5-3.0s滾動（頻率遞減）→3.5s碰牆叩（靜止）。其他環境聲降30%讓罐頭成焦點。

環境（美術）

狹窄水巷寬約2m向前約30m後拐彎消失。約10cm深積水——油光灰綠死水。水面漂紙片木屑塑膠袋。水面下沉底物：人字拖、梅林午餐肉罐、陽光檸檬茶泡脹紙包、港式八卦週刊褪色明星面孔、一把生鏽剪刀。天花板破裂水管滴水成節拍器。管線上方臘腸臘肉鹹魚串掛風乾。

傳送系統規格（程式）

指向：搖桿指向水面→光圈=水面漣漪效果（非UI圓圈）。色溫暖色2700K。油汙水面反射帶彩虹色干涉紋。預回饋：極微弱水面張力破裂聲。

傳送瞬間（0.3秒）：邊緣vignette收縮到70%→展開回100%。水平傳送無失重感。

落地視覺：水花從腳下爆開有方向濺射（水巷有微弱水流方向）。高度約15cm。某些碰牆壁出現新水痕沿油漆紋理下流。漣漪向四周擴散碰壁反射交叉。漂浮物被推動。3-4秒消散。

落地聲音：咚（100-200Hz,100ms）＋嘩（500-1500Hz,200ms）＋噼啪（2000-5000Hz,300ms散射）＋迴音沿水巷傳播反射。

落地觸覺：中等震動50ms「軟」質感（液體介質）＋0.3秒微弱持續震動（水流動模擬）。

操作流程（程式）

  • 步驟3 配重落下：水泥塊沿導軌下滑（鐵鏈嘩啦＋摩擦聲＋碰撞聲）→閘門被拉起（液壓緩衝咔嗡聲）→水從底部湧過另一側形成湍流帶走漂浮物→手柄腳下微弱衝擊震動

設計哲學

傳送移動（Teleport）為預設。玩家永遠不應該覺得自己在「使用遊戲功能」，而是在「踏出下一步」。每次傳送都是一個完整的微體驗——有預備、有過渡、有落地——整個過程約 0.5 到 1 秒但塞滿感官資訊。

三階段回饋系統

第一階段：指向預回饋

  • 金屬地面：金屬光澤光圈 + 叩擊預告音效
  • 水面：漣漪效果 + 水面張力聲
  • 木板：暖色調光圈 + 吱嘎預告
  • 不穩定結構：光圈抖動 + 低頻應力聲

第二階段：傳送過渡

  • 水平傳送：邊緣 vignette + 短促手柄震動
  • 向下傳送：失重暗示（畫面微上飄）+ 落地重震
  • 向上傳送：壓縮展開感 +「拉起」持續震動

第三階段：落地環境反應（0.3 秒內即時回饋）

  • 金屬上：鏈條擺動、工具盒彈跳
  • 木板上：吱嘎微傾
  • 水中：水花漣漪、漂浮物被推動
  • 碎石上：細碎聲、石子滾動
  • 靠近環境物件時：杯子晃、牆板震、燈泡搖光影動

6.4 配音備註

阿徹需要一位能自然切換英文和粵語的配音演員（香港裔或混血背景）。關鍵要求：

  • 英文為主體語言——技術操作、劇情推進、黑色幽默全用英文
  • 粵語為情緒觸發語言——食物名稱、人名、粗口、最後一句切換
  • 切換必須自然——不是「現在我要說粵語了」而是語境觸發的自動切換
  • 黑色幽默的語氣：疲倦的、熟悉的、帶著老手的自嘲。不是快嘴喜劇——是慢燒的、乾的、在害怕的時候更乾
