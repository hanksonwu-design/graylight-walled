語言設計 Language Design

  • 角色：Archer（阿徹） — 香港裔混血，自然切換英粵
  • 英文為主體語言，表達主要敘事、觀察與自嘲
  • 粵語作為情緒觸發，在情感高點或文化記憶觸發時自動切換
  • 語調：疲倦、自嘲、慢燒、乾式幽默（dry humour）
  • DEMO 最後一句台詞為粵語：「給個準話。」— 代表情感累積的釋放

美術風格 Art Style

三層龐克（Punk）美學疊加：

  • 蒸氣龐克 Steampunk 殖民層 — 黃銅、花紋鑄鐵、花體英文、維多利亞渦卷、精密齒輪
  • 柴油龐克 Dieselpunk 城寨層 — 鑄鐵、大鉚釘、裸齒輪、廢料拼接、粗暴工業
  • 賽博龐克 Cyberpunk 資訊層 — CRT 雪花屏、LED 種菜、頻率紙卷

關鍵原則：三者通過接縫（焦點）相遇。蒸氣=殖民原裝優雅；柴油=城寨 DIY 粗獷；兩者並存時，接縫就是故事。

教學曲線 Teaching Curve (5 階段漸進)

  • Phase 1 (S0): 純感官 — 無互動，建立空間感與心理預期
  • Phase 2 (S1): 基礎手部互動 — 撕塑膠帶、推門，教學「材質影響觸感」
  • Phase 3 (S2): 傳送 + 簡單謎題 — 水巷傳送教學、閘門插銷配重、路線選擇
  • Phase 4 (S3): 核心工具 + 中等謎題 — 固定裝置教學、主迴路重啟
  • Phase 5 (S4-S5): 複合挑戰 — 雙窗口固定、連鎖傳送、啟動序列

程式 Programming ▶
互動 A: 撕塑膠帶 ×2
  1. 接近封鎖線（15cm 範圍觸發）→ 微振提示
  2. 握拳抓取
  3. 拉扯 — 繃緊吱吱聲 + 觸覺阻力漸增
  4. 啪！斷裂 — 碎片飄落粒子
教學設計：第一條塑膠帶有彈性（需要用力），第二條一拉就斷 — 教會玩家「材質影響觸感」。
互動 B: 推柵門
  1. 雙手抓橫桿（3cm 鏽蝕觸感）
  2. 前半段：巨大阻力，金屬刮水泥聲
  3. 鏽壓碎 → 阻力減 60%
  4. 順暢滑動 → 撞到門檔
  5. 罐頭掉落 4 秒音效鏈：0s 叮 → 0.3s 噹 → 0.5-3s 滾動 → 3.5s 碰牆叩
  6. 環境音降低 30%（對比反差）

程式 Programming ▶
首次傳送（全遊戲最重要的傳送）

傳送變體
  • 靠牆傳送：濺牆更多
  • 深水區傳送：更沉、更強回饋
  • 踩沉底物：硬物回饋
  • 乾燥處傳送：聲音突變（對比反差）
閘門謎題機制（約 3 分鐘）
  1. 找到鑄鐵插銷穿孔位
  2. 拔出插銷 — 冰冷粗糙觸感、強阻力、嘶聲
  3. 齒輪彈簧釋放 — 黃銅精密咔嗒嗒嗒
  4. 配重落下 — 鐵鏈嘩啦 + 摩擦 + 碰撞
  5. 閘門升起 — 液壓咔嗡
  6. 水流湧過帶漂浮物
閘門前後水體體驗不同：前方死水 vs 後方有流動。
